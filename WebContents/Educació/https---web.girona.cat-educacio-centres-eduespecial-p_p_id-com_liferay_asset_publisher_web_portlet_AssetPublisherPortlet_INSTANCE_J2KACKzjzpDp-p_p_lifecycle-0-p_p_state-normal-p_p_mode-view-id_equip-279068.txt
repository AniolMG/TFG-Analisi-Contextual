El Centre Públic d’Educació Especial Palau és un centre educatiu, situat al barri de Palau  que  dóna resposta educativa  a nens/es, nois/es de 3 a 21 anys de  diferents comarques gironines. Atén alumnes amb necessitats educatives especials derivades d’un trastorn neuromotor.
